## GDPR – základná praktická kontrola
Toto nie je právne poradenstvo, ale praktický checklist pre web.

Ak web obsahuje kontaktný alebo rezervačný formulár:
- [ ] pridať checkbox so súhlasom so spracovaním osobných údajov,
- [ ] vysvetliť, na čo sa údaje použijú,
- [ ] uviesť, kto údaje spracúva,
- [ ] uviesť kontaktný e-mail,
- [ ] vytvoriť stránku „Ochrana osobných údajov“.

Ak web používa cookies/analytiku:
- [ ] zistiť, či sa používajú analytické alebo marketingové cookies,
- [ ] ak áno, pridať cookies lištu,
- [ ] umožniť odmietnutie nepovinných cookies.

Ak web používa Google Maps embed:
- [ ] zvážiť jednoduchý odkaz na Google Maps namiesto priameho embed iframe, ak chceš minimalizovať cookies,
- [ ] alebo informovať o použití externej služby v ochrane osobných údajov.

Pri fotkách zákazníčok:
- [ ] mať súhlas so zverejnením,
- [ ] pri fotkách pred/po nerobiť neprimerané alebo zavádzajúce sľuby,
- [ ] nepoužívať fotky bez jasného povolenia.
